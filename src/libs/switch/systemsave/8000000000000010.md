## [Switch](../../formats.md#nintendo-switch) > [System Save](../systemsave.md) > 8000000000000010

This archive is mounted to `account:/` by the account sysmodule. It contains information about the user profiles on the system.

The following files can be found in this archive:

* `su/`
    * `avators/`
        * [`<id>.jpg`](#avatorsidjpg)
        * [`profiles.dat`](#avatorsprofilesdat)
    * `baas/`
        * [`<id>.dat`](#baasiddat)
    * `cache/`
        * `<id>.dat`
    * `nas/`
        * `<id>.dat`
        * `<id>_aux.dat`
        * `<id>_id.token`
        * `<id>_op2membership.dat`
        * `<id>_refresh.token`
        * `<id>_user.json`
    * [`registry.dat`](#registrydat)

## registry.dat
This file is encoded in big-endian byte order.

| Offset | Size | Description |
| --- | --- | --- |
| 0x0 | 8 | Always `29 01 85 BC 01 60 11 60` |
| 0x8 | 8 | Padding |
| 0x10 | 16 * 8 | User id (UUIDv1) of up to 8 users on the device |
| 0x50 | 16 | Unknown |

## avators/&lt;id>.jpg
This file simply contains the profile picture of the user in JPEG format. The id is the UUIDv1 of the user account, which can be found in [profiles.dat](#avatorsprofilesdat).

## avators/profiles.dat
This file contains information about user profiles on the Nintendo Switch.

| Offset | Size | Description |
| --- | --- | --- |
| 0x0 | 8 | Always `A4 87 1A CD 01 60 61 50` |
| 0x8 | 8 | Padding |
| 0x10 | 200 * 8 | [Profile entries](#profile-entry) |
| 0x650 | 8 | Always `A4 87 1A CD 02 30 81 80` |
| 0x658 | 8 | Padding |
| 0x660 | 256 * 8 | [Unknown entries](#unknown-entry) |
| 0xE60 | 8 | Always `A4 87 1A CD 02 31 21 90` |
| 0xE68 | 8 | Padding |
| 0xE70 | 272 * 8 | [Baas entries](#baas-entry) |
| 0x16F0 | 1920 | Unknown (always 0?) |

### Profile Entry
| Offset | Size | Description |
| --- | --- | --- |
| 0x0 | 16 | User id (UUIDv1) |
| 0x10 | 16 | Unknown (UUIDv1) |
| 0x20 | 8 | Last update time (timestamp) |
| 0x28 | 32 | Username (padded with zeros) |
| 0x48 | 128 | Unknown |

### Unknown Entry
| Offset | Size | Description |
| --- | --- | --- |
| 0x0 | 16 | User id (UUIDv1) |
| 0x10 | 240 | Unknown |

### Baas Entry
| Offset | Size | Description |
| --- | --- | --- |
| 0x0 | 16 | User id (UUIDv1) |
| 0x10 | 128 | Profile image URL |
| 0x90 | 128 | Unknown |

Example URL:

[https://cdn-image-e0d67c509fb203858ebcb2fe3f88c2aa.baas.nintendo.com/1/9e3347f5cb1de530](https://cdn-image-e0d67c509fb203858ebcb2fe3f88c2aa.baas.nintendo.com/1/9e3347f5cb1de530)

## baas/&lt;id>.dat
These files contain information about device accounts that are registered on the [BaaS server](github.com/kinnay/NintendoClients/wiki/BAAS-Server). The id is the user id, which can be found in [registry.dat](#registrydat).

| Offset | Size | Description |
| --- | --- | --- |
| 0x0 | 8 | Unknown |
| 0x8 | 4 | Unknown |
| 0xC | 4 | Unknown |
| 0x10 | 8 | Unknown |
| 0x18 | 4 | Unknown |
| 0x1C | 4 | Unknown |
| 0x20 | 8 | Baas user id |
| 0x28 | 40 | Baas user password |
