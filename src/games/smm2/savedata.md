## [SMM 2](../../formats.md#smm2) > Save Data

This page is based on Super Mario Maker 2 version 3.0.3.

For every saved course, the following files can be found in the save data archive:

* [`course_data_<id>.bcd`](level.md)
* `course_replay_<id>.dat`
* `course_thumb_<id>.btl`

The `id` consists of three decimal digits from `000` to `999`.

In addition, the following files are always present:

* [`later.dat`](savedata/later.md)
* [`myworld.dat`](savedata/myworld.md)
* [`network.dat`](savedata/network.md)
* [`quest.dat`](savedata/quest.md)
* [`save.dat`](savedata/save.md)
